-Creating a VPC using Terraform Modules
-Using Input variables for VPC and module and refrence them in VPC Terraform Module, and loacl variables

    #Terraform Validate
terraform validate

    #Terraform plan
terraform plan

    #Terraform apply
terraform apply -auto-approve
